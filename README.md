sota_client_cpp
------

C++ Implementation of SOTA Client


Dependencies
------


libboost-program-options1.58.0
cmake (3.5.1)

Building
------

To build the client:

~~~
cd code
cmake CMakeLists.txt
make
~~~

Testing
-----

~~~
make test
~~~
