sota_client_cpp
------

C++ Implementation of SOTA Client

Dependencies
------

libboost-dev
libboost-program-options-dev
cmake

libboost-version: 1.58.0
cmake-version: 3.5.1

Building
------

To build the client:

~~~
mkdir build
cd build
cmake ..
make
~~~

Testing
-----

To run the test suite:

~~~
make test
~~~
